# research/

Supervised analyses backing the production fatigue / health-KPI signal.

## Setup

```bash
cd research
uv sync
uv run jupyter lab
```

## Notebooks

- **`fatigue_kpi_research.ipynb`** — feature analysis, model selection and
  validation backing the per-creative fatigue classifier shipped in
  `backend/app/services/fatigue.py`. Open it for the full story; the headline
  numbers are summarised in the final cell.
